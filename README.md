# FaithCalibrator

> Helping families get calibrated in Christ

Biblical resources and devotionals for couples and families to deepen their spiritual walk through intentional simplicity and sacred practice.

## About
FaithCalibrator provides 12-week and 12-month devotional programs for:
- Engaged couples preparing for Kingdom marriage
- Newlyweds growing together
- Families building a Christ-centered home
